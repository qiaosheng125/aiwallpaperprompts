# 内容增长计划

更新时间：2026-06-01

## 当前阶段

站点已经具备基础内容站形态：

- 首页视觉图片墙
- 16:9 桌面壁纸分类
- 9:16 手机壁纸分类
- 每张图独立详情页
- prompt / negative prompt
- 4K 下载
- 点赞排序
- 搜索、比例筛选、标签筛选、排序
- R2 图片存储

下一步目标不是无限堆图，而是把图库做成“可浏览、可搜索、可持续扩张”的内容资产。

## 第一阶段：121 到 150 张

目标：补齐主要比例和基础分类。

建议新增：

- 16:9 桌面图：40-50 张
- 9:16 手机图：25-35 张
- 无人物环境图：15-25 张

优先分类：

- Desktop Wallpaper
- Mobile Wallpaper
- Impressionist Anime
- Gothic Anime
- Lo-fi Anime
- Pixel Art
- Starlight
- Night Sky
- Garden
- River
- City

要求：

- 每个重点分类至少 10 张。
- 9:16 分类至少 40 张后，手机入口才有足够浏览感。
- 每张图都保留 Image2 优化 prompt 和 negative prompt。
- 不上传未审核图片。

## 第二阶段：150 到 300 张

目标：形成可被搜索引擎理解的内容簇。

建议扩展内容簇：

- anime phone wallpaper prompts
- anime desktop wallpaper prompts
- gothic anime wallpaper prompts
- lo-fi anime room wallpaper prompts
- starlight anime wallpaper prompts
- impressionist wallpaper prompts
- no character AI wallpaper prompts

策略：

- 每个主风格 20 张以上。
- 每个主场景 20 张以上。
- 桌面和手机比例都覆盖热门主题。
- 分类页补 200-400 字说明和内部链接。

## 第三阶段：300 张以上

需要评估架构是否升级：

- 静态详情页仍可继续，但 sitemap 可能需要分片。
- 分类页要继续增强筛选和排序。
- 首页只展示精选、最新、热门，避免一次性加载太多。
- 下载、点赞、访问数据要进入复盘。

## 每日内容节奏

启动期建议：

- 每天审核 30-60 张。
- 每天上传 20-50 张精品。
- 每周至少复盘一次分类缺口。

稳定后：

- 保持日更，但不牺牲质量。
- 某个分类质量不足时宁可少发。
- 某个分类有流量后再集中补图。

## 上传边界

只有用户明确确认“这批图片可以上传”时，才能执行上传。

当前待上传目录：

```txt
E:\程序\2k4k\网站图片
```

上传后归档目录：

```txt
E:\程序\2k4k\已上传归档\aiwallpaperprompts\YYYY-MM-DD
```

## 免费与付费策略

同类竞品不是全部免费，但多数采用“免费入口 + 后置变现”：

- 纯壁纸下载站通常免费浏览和下载，主要依赖广告、App、素材包或会员高清包。
- AI 壁纸生成站通常是 freemium，免费给少量额度，高质量生成、更多次数或高级模型收费。
- Prompt 图库通常免费浏览一部分内容，会员解锁高级 prompt、私密生成、批量能力或更多 credits。

本站当前阶段继续免费：

- 免费浏览图片。
- 免费复制 prompt 和 negative prompt。
- 免费下载原图。
- 不上登录墙，不提前挡住核心价值。

原因：当前最重要的是收录、点击、复制、下载、停留和点赞数据。过早收费会降低 SEO 和用户行为数据质量。后续等自然流量和高频行为出现后，再考虑精选 prompt pack、批量下载、收藏夹、会员图库、Image2 专用高级 prompt 包或广告。

当前定位：已经生成好的 4K 成品图 + 可直接复用的 Image2 prompt + 免费浏览下载 + 持续更新。
